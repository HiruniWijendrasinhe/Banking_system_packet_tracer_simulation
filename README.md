# Enterprise Banking Network Simulation (Cisco Packet Tracer)

## 📌 Project Level
Intermediate – Advanced Networking Simulation

## 📖 Project Overview
This project simulates a **banking enterprise network infrastructure** using Cisco Packet Tracer.  
The network consists of a **Headquarters (HQ), two local branch offices, one overseas branch, and a centralized data center**.

All branches are connected through **ISP routers**, enabling secure communication between locations.  
The network design implements **dynamic routing, firewall security, and enterprise network architecture principles**.

The headquarters network follows a **3-tier architecture model (Core, Distribution, Access)** to represent a scalable enterprise infrastructure.

---

## 🏢 Network Architecture

The simulated enterprise environment contains:

- Headquarters (HQ)
- Two Local Branch Offices
- One Overseas Branch
- Centralized Data Center
- ISP routers connecting all sites

Communication between branches is achieved through **dynamic routing using OSPF**.

---

## ⚙️ Key Networking Technologies

- **OSPF Dynamic Routing**
- **Cost-based OSPF path selection**
- **Automatic route failover**
- **Access Control Lists (ACL) for firewall security**
- **Enterprise 3-Tier Architecture**
  - Core Layer
  - Distribution Layer
  - Access Layer

---

## 🔁 Network Redundancy

The network implements **cost-based OSPF routing**.

If a primary link fails, OSPF automatically selects the **next lowest-cost route**, ensuring **continuous connectivity between branches and the data center**.

This demonstrates **network resilience and fault tolerance**.
